Conclusion

My Reflection

It's high time I concluded my experience in this course and said what I have learned in a few words. I have confidently internalized what the data life cycle is, and which ethical considerations are relevant in each stage. I had fun writing the essays and researching sources dealing with ethics in the digital world. I have also got to discuss such topics with my peers, and then play around with the HTML and CSS code to create this website (haven't done that in a while) :D

Overall, information technologies require a special ethical approach. This unfathomable field spans over many social, cultural, and issues: copyright, consent, data privacy, data ownership, data laws, bias in data, and more! This course provided a solid base for understanding some of these intricacies.

Our trip ends here. Now, let's go back home.
