Week 4: Data Use & Reuse

This is the Part Where We Actually Use Data

One important idea from this week I'd like to mention is that, as a researcher or a company collecting data, you are obligated to initially declare the intended use of the data, and if you want to change that, you have to ask for users' consent again. You can only use the data for the stated purposes and nothing else. Contrary to this principle, Mozilla appears to have silently removed the statement about not selling users' personal data from their Q&A section for users, which is a very concerning move.

Well, the data has been used up and insights have been received. Time to publish them!
